元素显示模式

HTML元素 一般分为 块元素 和 行内元素 一、块级元素的特点

  1. 自己独占一行
  1. 高度、宽度、外边距和内边距都可以控制
  1. 宽度默认是容器（父级宽度）的100%
  1. 是一个容器及盒子，里面可以放行内或块级元素

    #<p> <h>主要用于存放文字，所以不能放块级元素

    #转换成行内元素display: inline (较少使用)

二、行内元素的特点

  1. 相邻的元素在一行上，一行上可以显示多个
  1. 高、宽直接设置是无效的
  1. 默认宽度就是它本身内容的宽度
  1. 行内元素只能容纳文本或其他行内元素

    #链接里不能再放链接

    #特殊情况<a>里可以放块级元素，转换成块级模式display: block最安全

三、行内块元素 <img> <input> <td>

  1. 一行可以显示多个，但是之间会有空白缝隙
  1. 默认宽度就是本身的宽度
  1. 高度、宽度、外边距和内边距都可以控制

    #转换成行内块元素 display: inline-block

选择器

基础选择器

  1. 标签选择器
  1. 类选择器 class="XXX" / .XXX { / 多类名：<div class="red font35">
  1. id 选择器（一般和js搭配） id="XXX"/ #XXX {
  1. 通配符选择器（选取页面中所有元素）* {

复合选择器

  1. 后代选择器（选择父元素里的子元素）ol li a { .nav li a {
  1. 子选择器（必须选择最近一级的子元素) .nav > p {
  1. 并集选择器 div, p { div, p, .pig li { (一般竖着写)
  1. 伪类选择器（添加特殊效果）
1.链接伪类

a:link  选择所有未被访问的链接
a:visited  选择所有已被访问的链接
a:hover  选择鼠标指针位于其上的链接
a:active  选择活动链接（鼠标按下未弹起）

#为了确保生效，按LVHA顺序声明
#a链接必须要单独指定样式，默认样式不起作用

2.focus伪类选择器（选取获得焦点的表单元素）

input: focus

属性、值和单位

字体

1.字体系列

font-family:Arial, "Microsoft Yahei", "微软雅黑", serif;

#如果多个单词组成的字体，加引号
# monospace (usually for code) 

2.字体大小

Font-size:100% = 16px
Font-size:1em = 16px (em 代表一个文字大小)
Font-size:1rem = ignore the parent size

#标题标签要单独指定文字大小，不能在body里改

3.字体粗细

font-weight:normal或写400 / bold或写700 / bolder / lighter 

4.斜体

font-style: normal / italic (斜体) /

5.复合属性

font: font-style font-weight font-size/line-height font-family
例：font: italic 700 16px/20px "Microsoft Yahei";

#不能颠倒顺序
#不需要的属性可以省略（取默认值），但必须保留font-size和font-family属性

文本

1.文本颜色 color: red / #XXXXXX / rgb(XXX,XXX,XXX);
2.对齐文本(水平) text-align:left / right / center;
3.装饰文本 text-decoration: none / underline / overline (几乎不用) / line-through
4.文本缩进 text-indent: 2em; (只缩进首行)
5.行间距 line-height: 26px / 2

#行间距=上间距+文本高度+下间距 (可以从上一行底测量到下一行底即是行间距)
#单行文字垂直居中：让文字的行高等于盒子的高度
#想要图片居中对齐，则是让它的父亲P标签添加水平居中的代码

背景

1.背景颜色 background-color: transparent (默认) / color
2.背景图片 background-image: none / url()  #logo或小的装饰性的图片，便于控制位置
3.背景(图片)平铺 background-repeat: repeat (默认) / no-repeat / repeat-x / repeat-y

#背景图片会压住背景颜色

4.背景图片位置 background-position: x y

#xy可以使用方位名词或精确单位，可以混合使用
#如果是方位名词，前后两个值顺序无关，效果一致 例如：left top / top left
#如果只指定了一个方位名词，另一个省略，则默认(y轴)居中对齐

5.背景附着 background-attachment: scroll (默认) / fixed
6.复合写法 background:   

#没有特殊顺序，一般为颜色、图片、平铺、滚动、位置

7.背景颜色半透明 background: rgb(0,0,0,0.5);

#最后一个参数为alpha透明度，取值范围在0-1之间，0为100%透明，0.5的0可以省略
#CSS3 新增属性 IE 9+
#背景半透明，内容不受影响

The border-style property may be specified using one, two, three, or four values.

  • When one value is specified, it applies the same style to all four sides.
  • When two values are specified, the first style applies to the top and bottom, the second to the left and right.
  • When three values are specified, the first style applies to the top, the second to the left and right, the third to the bottom.
  • When four values are specified, the styles apply to the top, right, bottom, and left in that order (clockwise).
